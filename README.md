# RYANTRUST-NV

Production-oriented static website for **RYANTRUST — Trusted Counsel. Lasting Legacies.**

## Run locally
```bash
python3 -m http.server 8000
```

## Build
```bash
python3 scripts/build.py
```

## Launch blockers
1. Connect the consultation form to a secure, approved intake endpoint.
2. Confirm attorney biography, bar status, trade-name use, office eligibility, and all advertising disclosures.
3. Configure privacy-conscious analytics and update the privacy notice to match.
4. Replace the map placeholder with an approved map link/embed.
5. Run accessibility, link, structured-data, and cross-device QA.
6. Configure `ryantrust.com` in Vercel and DNS.
